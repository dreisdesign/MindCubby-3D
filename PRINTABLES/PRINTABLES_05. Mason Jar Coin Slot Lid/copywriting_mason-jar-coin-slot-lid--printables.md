### Description

**Design, Function, and Customization**

This is an original 3D printed coin slot lid designed to screw tightly onto standard mason jars. The lid is precision measured for a secure fit and has been tested for reliable closure. Perfect for saving coins or creating a simple bank.

- Modular Assembly: Prints as a single piece, no supports required.
- Precision Fit: Engineered for standard mason jar threads.
- Easy Use: Just screw on and start saving coins.
