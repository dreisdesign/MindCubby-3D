### Description

**Design, Function, and Customization**

Golf-ball-sized holder inspired by the iconic iMac G4 design. Features a snug grip for Apple Pencil and an embossed Apple logo. Prints as a single, no-assembly-required piece.

- **Retro iMac G4 Aesthetic:** Rounded form channels early-2000s Apple design.
- **Engineered Fit:** Snug interior grip secures pencil without slipping.
- **Embossed Logo:** Premium detail on rounded base.
- **No Supports:** Prints complete in one piece.
- **Smooth Surface Design:** Optimized to minimize layer stepping for a refined finish.
- **Minimal Material:** Fast print (~2.2 hours, 4.3 m filament).

### Print Tips & Settings

| Setting | Value |
| :---- | :---- |
| **Print Time** | ~2.2 hours |
| **Filament Used** | 4.3 m (~12 g) |
| **Layer Height** | 0.12 mm (fine detail) / 0.2 mm (standard) |
| **Nozzle Temp** | 195 °C |
| **Bed Temp** | 65 °C |
| **Printer** | Creality Ender-3 |
| **Slicer** | Cura 5.10.1 |
| **Model Dimensions** | ~60 × 60 × 25.5 mm |
| **Layer Count** | 210 (0.12 mm) / 135 (0.2 mm) |
| **Infill** | 15%, 30%, 50% options included |
| **Supports** | Not required |

Multiple print configurations provided: 0.12 mm (fine, ~2.2 hours) and 0.2 mm (standard, ~1.4 hours) layer heights with 15%, 30%, and 50% infill density options. Design optimized for smooth surfaces and minimal layer stepping. Tight tolerance on pencil grip is intentional for secure fit.

**Configuration Guide**

Choose your configuration based on desired surface smoothness, print time, and weight (structural rigidity):

| Configuration | Layer Height | Infill | Print Time | Filament | Weight | Use Case |

| **Fine Detail** | 0.16 mm | ~20% | ~1.9 hours | 4.6 m (~14 g) | Light & balanced | Best of both: smooth & quick |

### Assembly Instructions

No assembly required. Insert Apple Pencil into the top opening; the snug grip holds it securely in place.
