## üéØ What to Read First

### If you're new to the repository:
1. Read `README.md` in root directory (main overview)
2. Read `QUICK_REFERENCE.md` (this folder) for quick tasks
3. Read `PRINTER_SPECS.md` to understand your printer

### If you're setting up a profile:
1. Go to `../PROFILES/README.md` for profile usage
2. Check `PRINTER_SPECS.md` for material temperatures
3. Compare against `../PROFILES/Ender3V2_Baseline_StartGCode.gcode`

### If you're optimizing G-code:
1. Review `../PROFILES/Ender3V2_Baseline_StartGCode.gcode` (baseline)
2. Analyze current profile against baseline
3. Use `COPILOT_GUIDE.md` to document changes
4. Update `CHANGELOG.md` in root directory

### If you're troubleshooting:
1. Check `QUICK_REFERENCE.md` for quick fixes
2. Read `TROUBLESHOOTING.md` (when available) for detailed solutions
3. Use Copilot chat to ask: "Why is [issue] happening?"
