# G-Code Profiles & Optimization

Directory for G-code start/end sequences and optimizations.

## Files

### Archive
- `Original_StartGCode_Archive.gcode` - Original baseline (reference only)

### Active Optimizations
- `Optimized_StartGCode_v1.gcode` - Improved priming with 2-line sequence

## Organization

G-code files are named with version numbers to track optimization progress:
- `Original_*` - Baseline reference files
- `Optimized_*_v[N]` - Optimization iterations
